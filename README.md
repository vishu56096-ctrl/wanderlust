 Wanderlust (Airbnb Clone)

Wanderlust is a full-stack web application where users can explore, create, and manage property listings.

##  Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap

##  Features
- User Authentication (Login/Signup)
- Add, Edit & Delete Listings
- Image Upload
- Search Functionality
- Responsive UI

##  Live Demo
https://wanderlust-qov2.onrender.com/listings

This project helped me understand real-world full stack development, backend logic, and deployment.
